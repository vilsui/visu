Tiivistelmät
Karvinen 2026: SSH public key - Login without password
- SSH-avainpari muodostuu julkisesta ja yksityisestä avaimesta
- Julkinen avain kopioidaan palvelimelle jolloin salasanaa ei tarvita kirjautuessa
- Edellytys sujuvammalle automaatiolle kuten Ansiblelle

Oma huomio: Yksityinen avain on huomattavasti monimutkaisempi kuin tavallinen salasana,
mikä tekee siitä turvallisemman.

### Karvinen 2026: Hello Ansible
- Ansible on IaC-työkalu jolla voidaan hallita useita palvelimia kerralla SSH:n kautta
- Käyttää YAML-pohjaisia playbookkeja tehtävien määritykseen
- Ei tarvitse agentteja, mikä säästää resursseja ja vähentää tietoturvariskejä

Oma huomio: Idempotenssi tarkoittaa että saman playbookin voi ajaa monta kertaa,
mutta muutoksia tehdään vain jos niitä tarvitaan.

b) Pubkey

Automatisoin SSH-kirjautumisen julkisella avaimella.

Loin SSH-avainparin:

    ssh-keygen
    Generating public/private ed25519 key pair.
    Enter file in which to save the key (/home/visu/.ssh/id_ed25519):
    /home/visu/.ssh/id_ed25519 already exists.
    Overwrite (y/n)? y
    Your identification has been saved in /home/visu/.ssh/id_ed25519
    Your public key has been saved in /home/visu/.ssh/id_ed25519.pub

 
Kopioin julkisen avaimen localhostiin:

    $ ssh-copy-id visu@localhost
    /usr/bin/ssh-copy-id: INFO: 1 key(s) remain to be installed
    visu@localhost's password:
    Number of key(s) added: 1

 
Siirryin tarkistamaan että kirjautuminen onnistuu ilman salasanaa:
komennolla ssh localhost ja näytti toimivan niin kuin piti

 
c) Hei Ansible

Asensin Ansiblen ja tein hei maailma -playbookin.

sudo apt install ansible -ym nutta ansible minulla olikin jo asennettuna.
 

Loin kansion ja inventory-tiedoston:

    mkdir ansible
    cd ansible
    nano hosts



Hosts-tiedoston sisältö cat komennolla:

    cat hosts
    [local]
    localhost ansible_connection=ssh

 

Loin playbook-tiedoston:

    nano hello.yml

Hello.yml-tiedoston sisältö:

    cat hello.yml
 
    - name: Hei maailma
      hosts: local
      tasks:
        - name: Tulosta viesti
          debug:
            msg: "Hei maailma!"

 

Ajoin playbookin:

    ansible-playbook -i hosts hello.yml
    
    TASK [Tulosta viesti]
    ok: [localhost] => {
        "msg": "Hei maailma!"
    }
    
    PLAY RECAP
    localhost : ok=2 changed=0 unreachable=0 failed=0
